# Nepali Directory — Design System

## Brand
Nepal's trusted local business directory. Trusted, informative, community-first.

## Color Palette
- **Primary accent**: #FFD400 (yellow) — used for CTA buttons, highlights, logo badge, star ratings
- **Surface / background**: #ffffff (white), #f7f8fa (light gray page background)
- **Text primary**: #1a1a1a (near-black)
- **Text secondary**: #4b5563
- **Text muted**: #6b7280, #9ca3af
- **Border**: #e3e5e8, #d3d6db
- **Link**: #1a6dcc
- **Success / open**: #1a7d3a
- **Error / closed**: #b3261e
- **Yellow tint bg**: #fff7d1

## Typography (Manrope only)
- Display / Hero H1: 52–60px, weight 800, letter-spacing -1.5px
- Section H2: 24–32px, weight 800, letter-spacing -0.5px
- Card H3: 18–20px, weight 700
- Body text: 14–16px, weight 400–500, line-height 1.45
- Labels / tags: 11–13px, weight 600–700, often uppercase + letter-spacing
- Breadcrumb: 12px, weight 400, color #6b7280

## Header (Sticky, always present)
- White background, 1px bottom border #e3e5e8, sticky top
- Logo: 'ND' in #FFD400 yellow square (border-radius 3px) + 'Nepali / Directory' wordmark
- Dual search bar: [Find a business] | [Location] | [FIND yellow button]
- Right nav: Advertise · Write a Review · Log In · Sign Up (black pill)

## Layout
- Max-width 1180px centered, 24px horizontal padding
- Breadcrumb bar: #f7f8fa bg, 1px bottom border, 12px font, muted gray
- Cards: white bg, 1px border #e3e5e8, border-radius 8px, subtle hover shadow
- Section headings with 'View all →' links on right
- Grid-based layouts (repeat columns), gap 16–24px

## Buttons
- Primary CTA: #FFD400 bg, #1a1a1a text, bold, border-radius 4px, 11px 24px padding
- Secondary/black: #1a1a1a bg, #fff text, border-radius 4px
- Outline: white bg, 1px border #d3d6db, border-radius 4px
- Text link: #1a6dcc, underline on hover

## Components
- Star ratings: filled yellow #FFD400 stars, count in muted gray
- Badges: small pill with background, font-size 11px, weight 700
- Chips/tags: border 1px, border-radius 20px, font-size 12px
- Province/category pills: #fff7d1 bg + #FFD400 border for active
- Footer: dark #1a1a1a bg, white text, 4-col layout, ND logo in yellow

## Design Quality
Every page must have: sticky header, breadcrumb, rich hero or page header, dense information architecture, sidebar or tabs for navigation, footer. Modern 2026 premium quality — not generic. Think Yelp + Yellow Pages + local charm.
